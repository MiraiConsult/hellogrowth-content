# Criação Conteúdo LinkedIn

Luna LinkedIn, com base no ângulo selecionado pelo usuário e na mesma notícia utilizada para o Instagram, crie um post completo para LinkedIn.

## Tarefas a executar

### Post LinkedIn
- Criar um post com **1.000 a 1.500 caracteres**
- Estrutura recomendada:
  - **Gancho** (primeira linha que aparece antes do "ver mais") — máximo 150 caracteres, impactante
  - **Contexto** — apresentar a notícia ou dado relevante (2-3 frases)
  - **Análise/Opinião** — perspectiva profissional sobre o tema (3-5 frases)
  - **Insight prático** — o que o leitor pode aprender ou aplicar (2-3 frases)
  - **CTA** — pergunta ou convite para discussão nos comentários
- Incluir **3 a 5 hashtags** relevantes para LinkedIn (mais profissionais e específicas)
- Sugerir o **melhor horário de publicação** (com base nas melhores práticas do LinkedIn)

## Orientações gerais
- Tom de voz: profissional, analítico, com autoridade no assunto
- Adaptar a linguagem para o público do LinkedIn (mais formal que Instagram)
- Focar em insights de mercado, dados e análises
- Consultar o arquivo tone-of-voice.md para referência
- Consultar quality-criteria.md para padrões de qualidade
- Evitar linguagem excessivamente informal ou uso de emojis em excesso
