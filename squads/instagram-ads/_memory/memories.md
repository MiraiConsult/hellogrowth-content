- Preço transparente no CTA (R$99,90/mês) — reduz fricção

- image-generator precisa de OPENROUTER_API_KEY configurada — sem ela, usar gradiente CSS
- Chrome headless funciona com --no-sandbox para renders diretos
- Carousel overview (6 slides em grid) é útil para aprovação rápida
