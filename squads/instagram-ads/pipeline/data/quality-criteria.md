# Critérios de Qualidade: Anúncios Instagram Feed

## Critérios de Copy (peso: 60%)

### 1. Hook / Scroll-Stop (peso: 15%)
- [ ] O primeiro elemento visual + texto faz parar o scroll?
- [ ] O hook usa um driver emocional claro (medo, curiosidade, aspiração)?
- [ ] Os primeiros 125 caracteres do primary text entregam a proposta de valor?
- Benchmark: CTR > 0,88% indica hook eficaz

### 2. Proposta de Valor (peso: 10%)
- [ ] O benefício principal está claro e específico (não genérico)?
- [ ] Usa números e dados concretos quando possível?
- [ ] Diferencia HelloGrowth dos concorrentes?

### 3. CTA (peso: 10%)
- [ ] O CTA é específico e orientado à ação?
- [ ] A intensidade do CTA corresponde ao estágio do funil?
- [ ] Há exatamente UM CTA por anúncio (não múltiplos)?

### 4. Framework de Persuasão (peso: 10%)
- [ ] Um framework identificável foi aplicado (AIDA, PAS, BAB, 4Ps)?
- [ ] O framework está corretamente executado na sequência?
- [ ] O driver psicológico dominante é consistente do hook ao CTA?

### 5. Brand Voice (peso: 10%)
- [ ] O tom é consistente com HelloGrowth (direto, orientado a resultados)?
- [ ] Usa vocabulário da marca (estratégia, resultado, dado, comprovado)?
- [ ] Evita vocabulário proibido (hack, truque, segredo, guru, fórmula mágica)?

### 6. Técnica de Copy (peso: 5%)
- [ ] Frases curtas e diretas? Sem paredes de texto?
- [ ] Nenhum clichê da lista de anti-patterns?
- [ ] Copy poderia ser falada em voz alta sem soar artificial?

## Critérios Visuais (peso: 40%)

### 7. Layout e Formato (peso: 10%)
- [ ] Formato 4:5 (1080x1350px) ou formato adequado ao placement?
- [ ] Hierarquia visual clara: headline em destaque, supporting text menor?
- [ ] Contraste suficiente para legibilidade mobile?

### 8. Identidade Visual (peso: 10%)
- [ ] Cores da marca HelloGrowth aplicadas corretamente?
- [ ] Tipografia consistente com a identidade?
- [ ] Logo/marca presente sem dominar o layout?

### 9. Coerência Copy-Visual (peso: 10%)
- [ ] O visual reforça a mensagem da copy (não contradiz)?
- [ ] O gancho visual é tão forte quanto o gancho textual?
- [ ] O CTA visual é claro e visível?

### 10. Adequação à Plataforma (peso: 10%)
- [ ] Respeita as restrições do Instagram (sem links na copy)?
- [ ] Texto no criativo ocupa menos de 20% da área (regra Meta Ads)?
- [ ] Funciona bem em visualização mobile (93% dos usuários)?

## Regras de Decisão

| Condição | Veredicto |
|---|---|
| Média geral ≥ 7/10, nenhum critério < 4 | APROVADO |
| Média geral ≥ 7/10, critério não-crítico entre 4-6 | APROVADO COM RESSALVAS |
| Média geral < 7/10 | REJEITADO |
| Qualquer critério < 4/10 | REJEITADO (trigger automático) |
| 3+ ciclos de revisão com mesmos problemas | ESCALAR para o usuário |
